== Description ==

Quran Radio Widget is the first WordPress plugin that allows you to add a widget that plays an online Radio station
for the translation of the meaning of the Quran in more than 35 languages.


Major new features in Quran Radio 1.0 include:

* You can play 1 out of 35 translations of the Quran
* You can add the player to the Widget or to the post
* The player works 24/7
* You can control the display of the player

PS: For Support click to [Islam.com.kw](http://www.islam.com.kw/support/) .

== Installation ==

* Upload the Quran Radio plugin to your blog, Activate it.
* Go to Quran Radio from menu.
* From page select your language, choose icons and click Update options.
* Copy blue code like this: radio[*] and past code in your post or page.
* You can used radio in widget.
* Done!
